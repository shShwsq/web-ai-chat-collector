召回设置

控制 AI 问答 / 整理信息 / 生成测验时如何从向量库检索相关片段

先按 Top-K 召回候选，再过滤掉相似度低于阈值的结果
从向量库中召回相似度最高的前 N 条候选
相似度低于此值的候选会被丢弃。建议 0.2-0.5；过低会引入噪声，过高可能漏召回
喂给 LLM 的单对话内容上限。命中消息取完整内容，超此上限会截断。建议 5000-10000；过小会截断长消息，过大可能超出模型上下文窗口

▼如何在 DevTools 中查看
  1. 点击上方「打开扩展页」按钮（或访问 chrome://extensions）
  2. 找到「ai-chat-collector」扩展，点击详情中的「Service Worker」链接
  3. 在弹出的 DevTools 窗口中切换到 Application 标签
  4. 左侧展开 IndexedDB，即可看到 AIChatCollector 和 AIChatEmbeddings 两个数据库
  5. 展开数据库可查看各 objectStore 的内容；右键可刷新或清空
